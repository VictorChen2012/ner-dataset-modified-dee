"SINGGALANG": an Indonesian NER dataset that automatically labeled
==================================================================

This dataset was generated automatically using method explained in Alfina et al. (2017). 
It used expanded DBpedia of MDEE_Gazetteer as the reference to label the token and contains 48,957 sentences.

The dataset was named SINGGALANG, a name of a mountain in West Sumatera, Indonesia.

Please cite our paper (Alfina et al, 2017) if you use this dataset for your publication.

Warning: It will require huge resources to train a NER model using this dataset ;-)
